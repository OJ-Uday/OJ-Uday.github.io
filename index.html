01 Systems I've shipped

vetlock — supply-chain behavioral diff

Reads two lockfiles, fetches every changed package, safely extracts each tarball, and reports capability deltas: new install scripts, new network endpoints, env-token harvest, obfuscation jumps, integrity tampering. 12 of 13 real npm supply-chain attacks caught by the replay corpus. 31 red-team exploits closed across a six-agent adversarial review. Ships with a NEVER-EXECUTE canary — the analyzer never runs code from analyzed packages, tested every build.

TypeScriptNodeSASTCI/CD

Operational-intelligence platform

Browser SDK → Go + ClickHouse backend → React dashboard, across a federated micro-frontend estate of 5+ applications. Two-tier element identity and capture-time causality feed an inference engine that names the UI automatically — analysts approve with one click, and a new app onboards with two HTML attributes, zero code changes.

TypeScriptGoClickHouseReact

Release-audit service

A flight recorder for deploys: an append-only ledger of everything that ships, joined against behavior data to diff before vs. after. Backtested across 1,648 deploys and 90 days of events; its first live run caught a same-day API regression at a 53% error rate — with session-level evidence attached.

GoClickHouseSQL

Zero-touch delivery

Merge to main and walk away: CI-authored image-bump PRs, second-identity approval, auto-merge landing, drift detection, pin-back rollback. The generic implementation is open source below as gitops-autopilot — a repo that deploys itself.

02 Open source

vetlock ↗

Behavioral diff for npm dependency updates — what does this update DO now that it didn't before? 17 detectors, HMAC-authed cache, Damerau-Levenshtein typosquat matching, GHSA adjacency, NEVER-EXECUTE invariant. 375 tests, 31 red-team exploits closed. npx vetlock demo tries it in 10 seconds.
