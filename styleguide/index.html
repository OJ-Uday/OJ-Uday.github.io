Ratios are computed with the WCAG 2.x relative-luminance formula against the current --color-bg. A badge marked AA clears 4.5:1 for body text; AAA clears 7:1. Alpha tokens are composited over --color-bg before measurement.

06 Cards & surfaces

Surfaces stack from --color-bg (page) → --color-surface (card) → --color-surface-raised (card on card) → --color-surface-high (nested inside raised).

.card

Standard surface. Border + radius + shadow-1. Hover flips the border to --color-accent.

.card.card--raised

Same shape, elevated background. Use inside another card or when you need visual weight without another border.

Nested surfaces

A raised card inside a card. Border tokens are alpha-tinted so they blend against any surface.

--color-bg
--color-surface
--color-surface-raised
--color-surface-high

07 Tags & badges

Tags describe (neutral pill on a raised surface). Badges assert state (filled, mono, uppercase). Choose by intent.

08 Code & kbd

Inline code--inline for short references inside body copy; .code for standalone blocks. Horizontal scroll on overflow so long lines don't stretch the layout. 

// Consume a semantic token, never a primitive.
button {
  background: var(--color-accent);
  color:      var(--color-on-accent);
  border-radius: var(--radius-md);
  transition: background var(--dur-fast) var(--ease-standard);
}

Press ? for shortcuts, or Ctrl + K to open the command palette. Reviewers press T to flip the theme.

10 Dialog native <dialog>

The .dialog class styles the native <dialog> element. Open via .showModal(); close on Escape or the close button.

A native dialog, tokenized

Shape from --radius-lg, elevation from --color-shadow-hover, backdrop from --alpha-black-50 + backdrop-filter. No custom overlay component required — the browser owns focus trap and Escape-to-close.

data-theme-scope="dark" authoritative baseline
data-theme-scope="light" scoped override
